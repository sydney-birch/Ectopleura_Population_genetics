# Larval Settlement Study Details and Workflow

We conducted a reciprocal transplant larval settlement study using larvae from 3 sites (CML: New Castle NH; York ME; Wells ME) and biofilm from the same 3 sites. 

The experiemental design is a 3x3 factorial CRD with 4 replicates. We used the same larval settlement methods that I previously established [here](https://onlinelibrary.wiley.com/doi/full/10.1111/mec.16968). A power calculation was completed for the number of replicates, we have 92% power at an alpha of 0.05. 

The input settlement data file used in R: Settlement_data_2024.csv

The Rscript used to analyze this data and make figures is: actinula_settlement_2024.R
